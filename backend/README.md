# تطبيقة تسيير الصفقات العمومية
# Application de Gestion des Marchés Publics

نظام إدارة المناقصات والمشاريع العمومية للمحاكم

Système de gestion des appels d'offres pour les tribunaux permettant aux 
ingénieurs de publier des projets et aux entreprises de soumettre leurs offres.

## المميزات / Fonctionnalités

### للمسؤولين / Pour les Administrateurs
- ✅ نشر مشاريع جديدة / Publier de nouveaux projets
- ✅ مراجعة العروض / Réviser les offres
- ✅ التحقق من الوثائق / Vérifier les documents
- ✅ قبول أو رفض العروض / Accepter ou rejeter les offres

### للمترشحين / Pour les Candidats
- ✅ تصفح المشاريع المتاحة / Parcourir les projets disponibles
- ✅ تقديم العروض / Soumettre des offres
- ✅ رفع الوثائق المطلوبة / Télécharger les documents requis
- ✅ متابعة حالة العروض / Suivre le statut des offres
